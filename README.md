# AI Video Generator (OpenCV Version)

## Overview
Generate high-quality animated videos using generative AI models without MoviePy. This project leverages OpenCV for video creation and Streamlit for an interactive interface.

## Features
- Download datasets from KaggleHub
- Generate AI-driven videos from textual prompts
- Customize style, FPS, and duration
- Save generated videos locally
- Streamlit-based UI
